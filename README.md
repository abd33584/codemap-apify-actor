### Using the Actor

1. **Open the actor** in Apify Console
2. **Enter repository URL**: `https://github.com/user/repo`
3. **Click "Start"**
4. **Get your codemap** in seconds!

### Input Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `repositoryUrl` | string | âœ… Yes | GitHub repository URL |
| `branch` | string | No | Branch to analyze (default: `main`) |
| `outputFormat` | string | No | `markdown`, `json`, or `both` (default: `markdown`) |
| `includeDependencies` | boolean | No | Show import relationships (default: `false`) |
| `languageFilter` | string | No | Filter by language (e.g., `python`, `javascript`) |
| `excludePatterns` | array | No | Additional glob patterns to exclude |
| `maxFileSize` | number | No | Skip files larger than X MB (default: `10`) |
